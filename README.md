# fight.me-frontend

This is a NextJS/React/Typescript/Tailwind project bootstrapped with `create-t3-app`.

## Quickstart

### Setup

```bash
    pnpm i
```

### Dev build

```bash
    pnpm dev
```

### Prod build

```bash
    pnpm build && pnpm start
```

### Unit tests

```bash
    pnpm test
```

### Storybook

```bash
    pnpm run storybook
```
